Open Music API Docs

Base Url

https://openmusic-api.dapi.my.id

Endpoints

Registration and Authentications Endpoint

Add New User

This API for adding new user

  • Method: POST

  • Path: /users

  • Headers:

    • Content-Type: application/json

  • Body Request:

    • username: string

    • password: string

    • fullname: string

If success the response should be return some data:

  • Status Code: 201

  • Body Response:

    • status: 'success'

    • data: { userId }

Add New Authentications

This API will generate access token and refresh token with JWT authentications

  • Method: POST

  • Path: /authentications

  • Headers:

    • Content-Type: application/json

  • Body Request:

    • username: string

    • password: string

If success the response should be return some data:

  • Status Code: 201

  • Body Response:

    • status: 'success'

    • data: { accessToken, refreshToken }

Update Access Token With Refresh Token

This API will update user expire access token with refresh token and return new access token for user

  • Method: PUT

  • Path: /authentications

  • Headers:

    • Content-Type: application/json

  • Body Request:

    • refreshToken: string

If success the response should be return some data:

  • Status Code: 200

  • Body Response:

    • status: 'success'

    • data: { accessToken }

Delete User Authentications

This API will delete refresh token on database

  • Method: DELETE

  • Path: /authentications

  • Headers:

    • Content-Type: application/json

  • Body Request:

    • refreshToken: string

Albums Endpoints

This API will create a new album on database

  • Path: /albums

  • Headers:

    • Content-Type: application/json

  • Body Request:

    • name: string

    • year: number

If success the response should be return some data:

  • Status Code: 201

  • Body Response:

    • status: 'success'

    • data: { albumId }

Get Album By Id

This API will return album by album id

  • Path: /albums/:id

If success the response should be return some data:

  • Status Code: 200

  • Body Response:

    • status: 'success'

  • Method: PUT

  • Path: /albums/:id

  • Headers:

    • Content-Type: application/json

  • Body Request:

    • name: string

    • year: number

If success the response should be return some data:

  • Status Code: 200

  • Body Response:

    • status: 'success'

    • data: { albumId }

Delete Album By Id

This API will delete album by id on database

  • Method: DELETE

  • Path: /albums/:id

Upload Album Cover

This API will upload and provide covers for album

  • Method: POST

  • Path: /albums/:id/covers

  • Headers:

    • Form: multipart/form-data

  • Body Request:

    • cover: file
      (MIME types from images and max file is 500KB)

Like Album

This API is a feature for liking albums

  • Method: POST

  • Path: /albums/:id/likes

Unlike Album

This API is a feature for unliking albums

  • Method: DELETE

  • Path: /albums/:id/likes

Get Number Of Likes Albums

If success the response should be return some data:

  • Status Code: 200

  • Body Response:

    • status: 'success'

    • data: { likes }

Add New Songs

This API will add a new songs on database

  • Body Request:

    • year: number

    • genre: string

    • performer: string

    • duration: number?

    • albumId: string?

  • Body Response:

    • data: { songId }

Playlists Endpoints

(Playlists are a limited resource. To access it requires an access token)

Create Playlist

This API will create playlist

  • Method: POST

  • Path: /playlists

  • Headers:

    • Content-Type: application/json

    • Authorization: Bearer {Access Token}

  • Body Request:

    • name: string

If success the response should be return some data:

  • Status Code: 201

  • Body Response:

    • status: 'success'

    • data: { playlistId }

Get Playlists

This API will get playlist that user owned or collaborated

  • Method: GET

  • Path: /playlists

  • Headers:

    • Authorization: Bearer {Access Token}

If success the response should be return some data:

  • Status Code: 200

  • Body Response:

    • status: 'success'

    • data: { playlists }

Delete Playlist By Playlist Id

This API will delete playlist by playlist id

  • Method: DELETE

  • Path: /playlists/:id

  • Headers:

    • Authorization: Bearer {Access Token}

Add Song To Playlist

This API will add a song to owned or collaborated playlist

  • Method: POST

  • Path: /playlists/:id/songs

  • Headers:

    • Content-Type: application/json

    • Authorization: Bearer {Access Token}

  • Body Request:

    • songId: string

Get Song In Playlist

This API will return a song data from playlist

  • Method: GET

  • Path: /playlists/:id/songs

  • Headers:

    • Authorization: Bearer {Access Token}

If success the response should be return some data:

  • Status Code: 200

  • Body Response:

    • status: 'success'

    • data: { playlist }

Delete Song In Playlist

This API will delete a song in playlist (only owner or collaborator can delete song in playlist)

  • Method: DELETE

  • Path: /playlists/:id/songs

  • Headers:

    • Authorization: Bearer {Access Token}

Get Playlists Activities

This API will return the history of adding or removing songs from a playlist by a user or collaborator

  • Method: GET

  • Path: /playlists/:id/activities

  • Headers:

    • Authorization: Bearer {Access Token}

If success the response should be return some data:

  • Status Code: 200

  • Body Response:

    • status: 'success'

    • data: { playlistId, activities }

Collaborations Endpoints

(Collaborations are a limited resource. To access it requires an access token)

Add Collaboration

This API will give access to the desired user to be able to access your playlist

  • Method: POST

  • Path: /collaborations

  • Headers:

    • Content-Type: application/json

    • Authorization: Bearer {Access Token}

  • Body Request:

    • playlistId: string

    • userId: string

If success the response should be return some data:

  • Status Code: 201

  • Body Response:

    • status: 'success'

    • data: { collaborationId }

Delete Collaboration

This API will remove playlist access permissions for the intended user

  • Method: DELETE

  • Path: /collaborations

  • Headers:

    • Content-Type: application/json

    • Authorization: Bearer {Access Token}

  • Body Request:

    • playlistId: string

    • userId: string

Export Endpoints

(Export are a limited resource. To access it requires an access token)

Export Songs In Playlist

This API will send a message to rabbitmq (as message broker) to send the song in playlist to targeted email (only playlist owner can export song in playlist)

  • Method: POST

  • Path: /export/playlists/:playlistId

  • Headers:

    • Content-Type: application/json

    • Authorization: Bearer {Access Token}

  • Body Request:

    • targetEmail: string
